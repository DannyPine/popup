# useClick

Adds click event listeners that change the open state (toggle
behavior).

```js /useClick/
import {
  useFloating,
  useInteractions,
  useClick,
} from '@floating-ui/react-dom-interactions';

// ...
const {context} = useFloating();
const {getReferenceProps, getFloatingProps} = useInteractions([
  useClick(context, {
    // props
  }),
]);
```

## Props

```js
interface Props {
  enabled?: boolean;
  pointerDown?: boolean;
}
```

### enabled

default: `true{:js}`

Conditionally enable/disable the hook.

```js
useClick(context, {
  enabled: false,
});
```

### pointerDown

default: `false{:js}`

Whether to prefer the `pointerdown{:.string}` event over
`click{:.string}` (for faster feedback). Keyboard clicks will
continue to work.
